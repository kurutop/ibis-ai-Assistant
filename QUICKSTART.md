# 🚀 Quick Start Guide

## เริ่มใช้งาน

```bash
cd ibis-v2
npm install
npm start
```

## คำสั่งพื้นฐาน

```
❖ Ibis ❯ /help          # ดูคำสั่งทั้งหมด
❖ Ibis ❯ /tools         # ดู tools ที่มี
❖ Ibis ❯ /status        # ดูสถานะ
❖ Ibis ❯ อ่านไฟล์ package.json ให้หน่อย   # ใช้งานตามปกติ
❖ Ibis ❯ exit           # ออกจากโปรแกรม
```

## ทุกอย่างพร้อมใช้แล้ว! ✅
